## On Telegram

<Steps>
  <Step title="Open the bot">
    [@usemeridianbot](https://t.me/usemeridianbot)
  </Step>
  <Step title="Send a command">
    - `/picks` — today's ranked shortlist
    - `/track` — the public track record
    - `/check <address>` — score any Solana token
  </Step>
</Steps>

## On the web

| Page | What it does |
|---|---|
| [`/demo`](https://meridian-udkd.vercel.app/demo) | Today's ranked shortlist + the live track record |
| [`/evaluate`](https://meridian-udkd.vercel.app/evaluate) | Paste any Solana token, get a full scorecard |

## For developers

The backend is a FastAPI service. Full schema is in the [API Reference](api-reference).

```bash
# Today's shortlist
curl https://meridian-backend-qae0.onrender.com/api/daily-shortlist

# Score any token
curl -X POST https://meridian-backend-qae0.onrender.com/api/evaluate \
  -H "content-type: application/json" \
  -d '{"token":"G7L2LRZyoE6FZgFo51Betj88UPMdnNi1iYmBrpfpswrm"}'

# Public track record (wins + misses)
curl https://meridian-backend-qae0.onrender.com/api/track-record
```

<Tip>
  All endpoints are public and read-only. The only protected one is `POST /api/run` (regenerates the daily shortlist; gated by a server-side secret).
</Tip>
