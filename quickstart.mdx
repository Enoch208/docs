<Tabs>
  <Tab title="Traders (60 seconds)">
    The fastest way in is Telegram.

    <Steps>
      <Step title="Open the bot">
        [@usemeridianbot](https://t.me/usemeridianbot)
      </Step>
      <Step title="Send a command">
        - `/picks` — today's ranked shortlist
        - `/track` — the public track record (hits + misses)
        - `/check <address>` — score any Solana token on the spot
      </Step>
      <Step title="(Optional) Open the web demo">
        [`/demo`](https://meridian-udkd.vercel.app/demo) shows the same data with the full market grid and an unlock flow for holders.
      </Step>
    </Steps>

    <Tip>
      Calls are framed as **worth investigating**, never **buy**. Every positive pick names its standout risk.
    </Tip>
  </Tab>

  <Tab title="Developers">
    Three public, read-only endpoints. No auth needed.

    <CodeGroup>
    ```bash curl
    # Today's shortlist
    curl https://meridian-backend-qae0.onrender.com/api/daily-shortlist

    # Score any token
    curl -X POST https://meridian-backend-qae0.onrender.com/api/evaluate \
      -H "content-type: application/json" \
      -d '{"token":"G7L2LRZyoE6FZgFo51Betj88UPMdnNi1iYmBrpfpswrm"}'

    # Public track record
    curl https://meridian-backend-qae0.onrender.com/api/track-record
    ```

    ```python python
    import httpx
    BASE = "https://meridian-backend-qae0.onrender.com"

    shortlist = httpx.get(f"{BASE}/api/daily-shortlist").json()
    print(shortlist["picks"][0]["composite_score"])

    pick = httpx.post(f"{BASE}/api/evaluate", json={
        "token": "G7L2LRZyoE6FZgFo51Betj88UPMdnNi1iYmBrpfpswrm"
    }).json()
    ```

    ```ts typescript
    const BASE = "https://meridian-backend-qae0.onrender.com";

    const shortlist = await fetch(`${BASE}/api/daily-shortlist`).then(r => r.json());

    const pick = await fetch(`${BASE}/api/evaluate`, {
      method: "POST",
      headers: { "content-type": "application/json" },
      body: JSON.stringify({ token: "G7L2…swrm" }),
    }).then(r => r.json());
    ```
    </CodeGroup>

    <Tip>
      Full schema in the [API Reference](api-reference) tab. It's pulled live from the FastAPI service's `/openapi.json` — every new endpoint shows up the moment the backend redeploys.
    </Tip>
  </Tab>

  <Tab title="Self-host">
    The bot is a thin client over the API — run it anywhere with Python and outbound HTTPS.

    ```bash
    git clone https://github.com/Enoch208/Meridian
    cd Meridian/backend
    python3 -m venv .venv
    .venv/bin/pip install httpx python-dotenv

    cat > .env <<EOF
    TELEGRAM_BOT_TOKEN=<your token from @BotFather>
    MERIDIAN_API_URL=https://meridian-backend-qae0.onrender.com
    EOF

    .venv/bin/python -m meridian.bot
    ```

    For an always-on deploy: see [Architecture → Running it yourself](architecture).
  </Tab>
