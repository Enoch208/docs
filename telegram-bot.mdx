The bot is a thin client over the Meridian API. DM it, get the call.

[**Open @usemeridianbot →**](https://t.me/usemeridianbot)

## Commands

| Command | What it does |
|---|---|
| `/start` | Intro + disclaimer |
| `/picks` | Today's ranked shortlist (top-3 + scout breakdown + reasons + risk) |
| `/track` | The public track record (hits, misses, open + recent calls) |
| `/check <address>` | Score any Solana token on the spot |
| `/help` | Same as `/start` |

## Example replies

`/picks`:

```
🛰 Today's shortlist · 2026-05-27

#1  $NOVA — score 87/100
  ✓ Mint & freeze authority renounced
  ⚠ Risk: Pair is young — unproven
  Clean signals across the board — worth investigating.

#2  $ORBT — score 82/100
  …
```

`/check <address>`:

```
🔎 $MRDN — score 31/100
Meridian

On-chain 85 · Liquidity 0 · Momentum 8 · Smart-money —

💧 $651 liq · 👥 19 holders · 24h −50%

⚠ Risk: Thin liquidity — exit-rug risk
Weak or unverified signals — investigate cautiously.

Worth investigating — never financial advice.
```

## What it doesn't do

The bot doesn't trade, doesn't recommend buys, and doesn't track wallets. It's a delivery surface for the swarm's calls — research, not advice.

## Run it yourself

The bot is open source. To run a private instance:

```bash
cd backend
.venv/bin/pip install httpx python-dotenv
echo "TELEGRAM_BOT_TOKEN=<your token from @BotFather>" >> .env
echo "MERIDIAN_API_URL=https://meridian-backend-qae0.onrender.com" >> .env
.venv/bin/python -m meridian.bot
```
