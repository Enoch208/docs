The bot is a thin Python client over the Meridian API. It long-polls Telegram, formats responses in HTML, and runs anywhere with outbound HTTPS. ~250 lines, no extra deps over the backend.

[**Open @usemeridianbot →**](https://t.me/usemeridianbot)

## Commands

| Command | What it does | Hits |
|---|---|---|
| `/start` · `/help` | Intro + disclaimer | none (static) |
| `/picks` | Today's ranked shortlist | `GET /api/daily-shortlist` |
| `/track` | Public track record (hits + misses + open) | `GET /api/track-record` |
| `/check <address>` | Score any Solana token | `POST /api/evaluate` |
| `/smart_money` | Top wallets in the [smart-money](smart-money) watchlist | `GET /api/smart-money/watchlist` |

Sends an instant *"scanning…"* / *"pulling…"* ack and **edits the same message** with the result — so a cold-starting backend never leaves the user staring at silence.

## Why a bot

Crypto alpha lives in Telegram. The daily call is a DM, not a tab to refresh. The bot exists because that's where the audience already is — and because chat is a natural delivery surface for a once-a-day signal.

The web `/demo` and the bot serve the **same data** through the same API; they're parallel surfaces, not hierarchical.

## Run your own

The bot is open source — run a private mirror in three steps:

```bash
git clone https://github.com/Enoch208/Meridian
cd Meridian/backend
python3 -m venv .venv
.venv/bin/pip install httpx python-dotenv

cat > .env <<EOF
TELEGRAM_BOT_TOKEN=<token from @BotFather>
MERIDIAN_API_URL=https://meridian-backend-qae0.onrender.com
EOF

.venv/bin/python -m meridian.bot
```

For always-on, use **pm2** on a VPS (cheapest), a **systemd unit**, or a Render background worker (paid tier). See [Architecture → Running it yourself](architecture).

## Caveats

- **Render free-tier cold start.** First request after ~15 min idle takes ~30s. The bot sends an instant ack so it doesn't feel dead.
- **No wallet connection in chat.** Hold-to-unlock is a web-only feature (Reown / Phantom). The bot delivers what the API exposes publicly — the daily shortlist, the track record, single-token evaluations. (Holder-gated content stays web-side.)
- **No financial advice. Ever.** Every reply ends with the disclaimer.
